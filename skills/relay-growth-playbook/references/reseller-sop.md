# Reseller SOP

## 1. 代理商能看到什么

- 自己名下账号和 key
- 自己可售商品
- 统一的价格口径和说明模板
- 充值引导和售后 FAQ

## 2. 代理商不能做什么

- 不直接修改总额度
- 不直接改系统倍率
- 不接触上游 key
- 不接触后台管理员入口

## 3. 代理资料建议内容

- 商品目录
- 基础价格逻辑
- 高质量版 / 混合版 / 全能版区别
- 接入方式
- 充值说明
- 常见问题
- 风险提示

## 4. 发卡与交付要点

- 发卡内容只写域名、Key、使用方法
- 默认说明预算限制和模型范围
- 额度不足时引导激活码或人工充值
- 不把后台截图中的敏感信息发给下游

## 5. 标准售后话术

至少准备三类标准回复：

- 新手接入不会配
- key 可用但模型不可用
- 额度不足需要充值
