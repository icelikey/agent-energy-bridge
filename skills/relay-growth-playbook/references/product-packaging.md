# Product Packaging

## 1. 商品分层

### 高质量版

- 面向：重度编码、复杂修复、高价值任务
- 卖点：成功率更高、复杂任务能力更强
- 风控：默认高价模型限制更严格

### 混合版

- 面向：普通用户、轻量自动化、日常办公
- 卖点：成本更友好，适合长期使用
- 风控：优先低成本模型，必要时升级

### 全能版

- 面向：新手、代理统一交付、多客户端兼容
- 卖点：一个入口适配多协议和多模型能力
- 风控：路由解释要清楚，避免用户误以为所有模型都等价

## 2. 对外必备字段

每个商品至少包含：

- 商品名
- 适用场景
- 接入协议
- 接入地址
- Key 使用方式
- 模型范围
- 预算限制
- 充值方式
- 风险提示

## 3. 表达原则

- 不承诺未验证的稳定性
- 不宣传不可用线路
- 不写源站 IP
- 不把“模型列表能看到”当成“模型真实稳定可用”

## 4. 推荐交付物

- 一页式接入说明
- 商品对比表
- FAQ
- 充值说明
- 代理商分发表
